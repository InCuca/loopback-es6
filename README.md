# loopback-es6
[![Build Status](https://travis-ci.org/InCuca/loopback-es6.png)](https://travis-ci.org/InCuca/loopback-es6)

A Loopback project scaffold using ES6, Babel, Gulp and Mocha for tests

## Installation

```
  $ npm install
```

## Testing

```
  $ npm test
```

## Running Development

```
  $ npm run dev
```

## Running Production (after build)

```
  $ npm start
```

## Build to ./build

```
  $ npm run build
```
